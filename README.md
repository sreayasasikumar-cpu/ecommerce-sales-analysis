# 📊 E-commerce Sales Analysis

## 📌 Project Overview

This project analyzes e-commerce sales data using Python for data processing, SQL for query-based analysis (demonstration), and Power BI for visualization to generate actionable business insights.

---

## 🛠 Tools Used

* Power BI (Dashboard & Visualization)
* Python (Pandas for Data Analysis)
* SQL (Query-based Analysis - Demonstration)
* Excel (Data Source)

---

## 🔄 Workflow

1. Data Collection (Excel dataset)
2. Data Cleaning using Python
3. Data Analysis using SQL queries (demonstrated through query scripts)
4. Data Visualization using Power BI
5. Business Insights Generation

## 📈 Key Insights

* The Technology category generates the highest sales, indicating strong demand in tech-related products.
* The West region contributes approximately **31% of total sales**, making it the top-performing region.
* Sales experienced a dip around **2015**, followed by consistent growth in later years.
* The overall profit margin is around **12%**, indicating moderate profitability.
* Furniture category shows lower profit compared to its sales, suggesting higher costs or inefficiencies.

---

## 📌 Recommendations

* Focus on expanding high-performing categories like Technology.
* Replicate successful strategies from the West region in other regions.
* Investigate the reasons behind the 2015 dip to reduce future risks.
* Improve cost efficiency in low-profit categories like Furniture.
